# 🤖 Smart AI Chatbot

## 📌 فكرة المشروع
A smart AI chatbot that can answer user questions using Google Gemini API.  
It provides a simple web interface for real-time interaction.

## 🚀 Features
- AI-powered responses
- Simple web interface using Gradio
- Fast and interactive

## 🛠️ Technologies Used
- Python
- Google Gemini API
- Gradio

## ▶️ How to Run

1. Open Google Colab
2. Install requirements:
   pip install google-generativeai gradio
3. Add your API Key
4. Run the notebook
